# Finance Analytics Dashboard - Power BI

## Project Overview

This project is an interactive Financial Analytics Dashboard developed using Microsoft Power BI. The dashboard provides insights into sales performance, profitability, discounts, units sold, and cost analysis across countries, products, and customer segments.

## Dashboard Features

* Interactive Filters (Country, Year, Product, Segment)
* KPI Cards
* Sales Trend Analysis
* Profit Analysis by Country
* Discount Analysis by Segment
* Product Performance Analysis
* Cost of Goods Sold (COGS) Analysis
* Country-wise Units Sold Treemap
* Landing Page Navigation

## Tools Used

* Microsoft Power BI
* DAX
* Microsoft Excel

## KPI Metrics

* Total Sales
* Total Profit
* Total Discounts
* Total Units Sold
* Total COGS

* Data Modeling
